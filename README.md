### [PL]
Wykrywanie twarzy odbywa się dzięki oprogramowaniu OpenCV![2]()
Pozostałe użyte technologie:
  - I2C
  - Serwomechanizmy
  - Czujniki na podczerwień
  

### Instalacja

```sh
$ git clone https://github.com/Dursky/Roboarm-Face-Detection.git
$ cd Roboarm-Face-Detection
$ python3 main.py
```
